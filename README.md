Мій перший власний платформер, написаний на Python (Pygame).
Проєкт переписано з використанням ООП та чистої архітектури.

## Як грати
- **Стрілки вліво/вправо**: Рух
- **Пробіл**: Стрибок (є подвійний стрибок)
- **Стрілка вгору**: Джетпак (витрачає паливо)
- **Z**: Ривок (Dash)
- **Shift**: Повільна хода

## Особливості
- Реалістична гравітація
- Система палива для польотів
- Збір монеток та лічильник очок
- Механіка платформ (можна стрибати знизу крізь них)

## Технології
- Python 3
- Pygame library
